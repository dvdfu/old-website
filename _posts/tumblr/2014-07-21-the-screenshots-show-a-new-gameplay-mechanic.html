The screenshots show a new gameplay mechanic tentatively known as wind shafts. Any movable piece in the same row or column is pushed in that direction, unless a non-movable piece is obstructing the source of wind. I took a break from regularly working on the game, but I have made (or am beginning to make) some major breakthroughs, one being clickable editor icons.

The game and its editor are playable here. Use Enter to switch between the editor/game, use Ctrl+Z or Ctrl+Y in the editor to undo moves. Click for everything else: the empty red square is tentatively the eraser. It’s probably useless having this released since no one knows how to play the game well enough to make levels, but there were a few things I had to test (for example, your created level will save between page visits).

Somehow, the editor icons took a while to implement. Right now most of my work is going into the game structure: the way the game switches between screens, applies input to active screens, and so on. It’s basic for sure, but definitely involves a lot of thinking.
